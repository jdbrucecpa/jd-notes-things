Material Design 3 Implementation Notes

  • Tonal Color System: Uses M3 dark theme palette with surface variants (#1c1b1f, #2b2930, #36343b) instead of pure grays for depth hierarchy
  • Primary Color: Lavender (#d0bcff) provides high contrast on dark surfaces and follows M3 accessibility guidelines
  • Rounded Corners: Full pill shapes (24-28px radius) for buttons create the friendly, approachable Material You aesthetic
  • FAB Treatment: Primary recording button uses large FAB with gradient and elevation shadow to establish clear visual hierarchy
  • State Layers: Interactive feedback uses +8% white overlay on hover, +12% on active (standard M3 opacity values)
  • Disabled State: Follows M3 guideline of 38% opacity, clearly indicating unavailable actions without removing context
  • Typography: Roboto font family with proper weight hierarchy (400 regular, 500 medium, 600 semibold, 700 bold)
  • Spacing: Consistent 8px grid system (8/16/24/32/48px) for predictable rhythm and alignment
  • Icons: 24x24 Material Icons style with rounded corners matching overall design language
  • Elevation: Uses tonal surfaces and subtle shadows instead of heavy drop shadows for modern, clean appearance
  • Badge Design: Tonal variant with high-contrast counter badge follows M3 notification patterns
  • Search Field: Filled text field variant with leading icon matches M3 input component specifications
  • Motion: Emphasized easing curve (cubic-bezier(0.4, 0, 0.2, 1)) at 200ms for responsive, polished interactions
  • Accessibility: Sufficient color contrast ratios, touch-friendly 40px icon button targets, clear focus states with 2px outline
